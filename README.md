# Carta del destino

해당 프로젝트는 Carta del destino(운명의 편지)라는 이름의 웹 애플리케이션입니다. AI 타로 리딩을 주요 목적으로 하며, 사용자가 온라인에서 쉽고 직관적으로 타로 카드를 뽑고, 인공지능 기반 해석을 받아볼 수 있도록 설계되었습니다. 운세, 자기성찰, 조언 등 다양한 주제에 맞춘 맞춤형 타로 경험을 제공합니다. React와 Node.js 기반의 풀스택 환경에서 동작하며, 사용자 인증, 데이터 관리, RESTful API, 테스트 자동화 등 다양한 기능을 지원합니다.

## 주요 기능
- 회원가입 및 로그인(JWT 인증)
- 사용자 정보 및 데이터 관리
- 프론트엔드와 백엔드 분리 개발
- RESTful API 제공
- 테스트 코드(Jest) 기반 품질 관리

## 기술 스택
- 프론트엔드: React, TypeScript, React Router, Axios, Jest
- 백엔드: Node.js, Express, TypeScript, JWT, MySQL, Sequelize, Jest
